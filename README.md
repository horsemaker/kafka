## Features

- User Authorization
    - Sign In
    - Sign Up
    - Sign Out
- Protected/Private Routes
- Rich Text Editor
- Add Note
- Edit Note
- Sort Notes
- Filter Notes
- Archive Note
- Delete Note

## Tech Stack

**Client:** React, React Router, Context API, Horsemaker's UI

**Server:** Mockbee (Mock Backend)


## Run Locally

Clone the project

```bash
  git clone https://github.com/horsemaker/kafka.git
```

Go to the project directory

```bash
  cd kafka
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
